Bot Overview

This Discord bot allows you to remotely control and monitor systems through Discord slash commands. The bot provides a secure way to manage your computers, view system information, capture screenshots, browse files, and execute commands - all through Discord's interface.

Getting Started

To use this bot, you need to:

  1. Invite the bot to your Discord server
  2. Use the /connect command to establish a connection
  3. You're ready to use all available commands!

Available Commands

Command Description
/connect Connect to a remote computer
/disconnect Disconnect from the current remote computer
/sysinfo Get detailed system information
/screenshot Take a screenshot of the remote system
/processes List top processes by memory usage
/kill <pid> Kill a process by PID
/ls [path] List files in a directory
/cd <path> Change working directory
/cat <file> View the contents of a file
/exec <command> Execute a shell command
/help Show all available commands
